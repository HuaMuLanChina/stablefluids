stablefluids
============

Author
------

Dongli Zhang (dongli.zhang0129@gmail.com): Fluid simulation

Contributor
-----------

Jerry Chen (onlyuser@gmail.com): Makefile & PNG support

Contents
--------

* GridStableFluid2D: Implemented with traditional grid.
* MacStableFluid2D:  Implemented with Mac grid.
* TextureFluid:      Implemented using a PNG image as texture.

Please refer [here](http://finallyjustice.github.io/fluid/) for related demos.

Reference
---------

1. Stable Fluids, Jos Stam, SIGGRAPH 1999.
2. Visual Simulation of Smoke, Ronald Fedkiw, Jos Stam, and Henrik Wann Jensen, SIGGRAPH 2001.
